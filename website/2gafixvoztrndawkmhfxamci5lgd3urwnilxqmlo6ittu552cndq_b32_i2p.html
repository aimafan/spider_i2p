Webogram (Telegram web client) в I2P 
2.10.18, 23:59 ( 5085 views ) - I2Pd, Сервисы - Posted by Administrator
http://re6cgwg2yrkgaixlqvt5ufajbb3w42fs ... a.b32.i2p/

Webogram работающий полностью через I2P. Не требует и не производит подключений в клирнет для работы с API Телеграма.
Все запросы направлены на вебсервер, который в дальнейшем проксирует запросы на официальные сервера.

Обмен данными производится по протоколу MTProto, так что параноики могут спать спокойно в шапочке из фольги.

Зеркало Флибусты 
1.10.18, 08:23 ( 6429 views ) - I2Pd, Сервисы - Posted by Administrator
Запушено неофициальное зеркало Флибусты.
http://xcktsvspaz4iu24g65vuzuvoagqzfilq ... a.b32.i2p/

Работает на nginx + i2pd.

Проксирование производится посредством изменения ссылок с помощью sub_filter и перезаписи перенаправлений через proxy_redirect.
Для корректной авторизации производится изменение поля domain в cookie с помощью proxy_cookie_domain.
