Notbob's toys

Webcounter for a static site

To use this just include it like you would any other image at the bottom of your page. But, be sure to change the number to a large random number. You can also change the color using a variety of color names. And the height as you see fit.


<img src="http://toys.notbob.i2p/counter.cgi?site=38238233&color=orange" height=40>

<img src="http://toys.notbob.i2p/counter.cgi?site=toys.notbob.i2p&color=red" height=20>

<img src="http://toys.notbob.i2p/counter.cgi?site=example.i2p&color=black" height=40>

<img src="http://toys.notbob.i2p/counter.cgi?site=justsomewords&color=green" height=60>

As you can see, you can also change the size. It should scale well for HDPI displays as it's a SVG file.
