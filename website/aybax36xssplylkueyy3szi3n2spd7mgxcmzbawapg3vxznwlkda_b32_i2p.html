Apophis [TRNG|HRNG] is a true random number generator that feeds off Galactic as well as atmospheric backround RF noise.

Signal processing flow consists of noise sampling, Von-Neumann debiasing and FIPS 140-2 test runs. Only data that have passed FIPS 140-2 tests are written to FIFO pipe. Computational services access this FIFO pipe in real-time and no data are being stored or kept back.

SYNOPSIS

[URL]/hexdump[?size={number of bytes}]

DESCRIPTION

Service provides random number stream in hexdump-C format.
If size field is not given, service returns 32KB stream.
Acceptable values for size are {0:64000}.

USAGE

http://services.apophis.i2p/hexdump?size=512

SYNOPSIS

[URL]/imagedump[?dim={image width in pixels}]

DESCRIPTION

Service provides visual representation of entropy received.
If dim field is not given, service returns 340x340 image.
Acceptable values for dim are {100:600}.

USAGE

http://services.apophis.i2p/imagedump?dim=500

SYNOPSIS

[URL]/bindump[?size={number of bytes}]

DESCRIPTION

Service provides downloadable random number binary file.
If size field is not given, service returns 32KB file.
Acceptable values for size are {0:64000}.

USAGE

http://services.apophis.i2p/bindump?size=18000

EXAMPLES

Generation of 80bit entropy passwords from binary file:

$ strings ./apophis.bin | grep -o '[a-hjkmnp-z2-9.]' |
   head -n 16 | tr -d '\n'; echo
