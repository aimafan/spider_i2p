Quicillith personal blog

Hello there!

You might be tempted to ask: "What's the point of having a blog on the darknet?"

I felt like a need some kind of space to express myself freely. A place which isn't a diary and where I can write about everything I have on mind.

I want it to be disconnected from my clearnet presence and identity as much as possible. I want to be able to share personal conclusions while maintaing a certain level of anonymity, which is such a rare thing these days.

Having summed up those criteria, I think that a darknet blog is a reasonable choice.

The fact that it's all in the Tor (and I2P) network means that only those who will receive a link will be able to access the site. This gives me the ability to share this with anybody while preserving obscurity from the public at the same time.

The site was published on the Tor network first, because it's accessible for an average user. ... or at least more accessible than I2P (which can be pretty tricky to setup).

I'll write stuff whenever I want to. There are no plans for any regularity. I'll write about all kinds of topics.

I come from a non-English speaking country but I'll write it all in English as it makes a great lingua franca worldwide.

Design principles

The website is expected to be fully operational without a single line of JavaScript, as it is known to pose security concerns.

The website shall respect the privacy of the viewer. No cookies. Anonimity of the viewers/commenters shall be respected as well.

The comment system is designed in such a way that it doesn't need to process any user data. Therefore, simple captchas have to be used to prevent abuse.

State of the blog

Currently, the blog is still at its' infancy, especially when it comes to content itself. The core infrastructure of the backend seems to be already there, although many new features are planned.

This is definitely not the final form.
