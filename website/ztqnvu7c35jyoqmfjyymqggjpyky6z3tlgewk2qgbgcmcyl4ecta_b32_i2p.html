    You can now leave notes on users!

    2022-10-03 12:00 UTC

    We've added the ability to post notes on a user. This ability is particularly useful for active traders that need to keep track of many different trading partners. If you, for example, offered a discount to a user for a future trade you'd be able to note that down for later reference. Or, perhaps, you need to keep track of the buyer's usual payment details to be aware of discrepancies. The notes can be set and are visible on any trade page with that user as well as on their user page. The notes are formatted in Markdown. These notes are confidential and only visible to you, they aren't visible to any other user.
